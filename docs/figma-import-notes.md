# Figma Import Notes

The architecture diagram is available at:

```text
docs/architecture-work-buddy.svg
```

To use it in Figma:

1. Open Figma.
2. Create a new design file.
3. Drag `architecture-work-buddy.svg` onto the canvas.
4. Ungroup or edit the imported vector layers as needed.
5. Export as PNG for Kaggle cover/writeup use.

Suggested Figma file name:

```text
Work Buddy - Architecture Diagram
```
